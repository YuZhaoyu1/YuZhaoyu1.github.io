520. 检测大写字母

题目要求

我们定义，在以下情况时，单词的大写用法是正确的：

  • 全部字母都是大写，比如 "USA" 。
  • 单词中所有字母都不是大写，比如 "leetcode" 。
  • 如果单词不只含有一个字母，只有首字母大写， 比如 "Google" 。

给你一个字符串 word 。如果大写用法正确，返回 true ；否则，返回 false 。

示例 1：

输入：word = "USA"
输出：true

示例 2：

输入：word = "FlaG"
输出：false

提示：

  • 1 <= word.length <= 100
  • word 由小写和大写英文字母组成

个人解答

  1. 直接模拟，用flag来记录是哪种情况，如果单词长度为1，那么肯定合格
  2. 如果首字母为大写，并且第二个字母也是大写，那么就是第一种，flag记为1
  3. 如果首字母大写，第二个字母小写，那么就是第三种，flag记为3
  4. 如果首字母小写，就是第二种情况，flag记为2
  5. 然后对后面的字母进行检查是否满足flag标记的情况，如果出现不满足，就返回false
  6. 全部检查完返回true
class Solution {
    public boolean detectCapitalUse(String word) {
        if(word.length() <= 1){
            return true;
        }
        int flag = 0;
        if(word.charAt(0) >= 'A' && word.charAt(0) <= 'Z'){
            if(word.charAt(1) >= 'A' && word.charAt(1) <= 'Z'){
                flag = 1;
            }
            else{
                flag = 3;
            }
        }
        else if(word.charAt(0) >= 'a' && word.charAt(0) <= 'z'){
            flag = 2;
        }
        for(int i=1;i<word.length();i++){
            if(flag == 1 && word.charAt(i) >= 'a' && word.charAt(i) <= 'z'){
                return false;
            }
            else if(flag == 2 && word.charAt(i) >= 'A' && word.charAt(i) <= 'Z'){
                return false;
            }
            else if(flag == 3 && word.charAt(i) >= 'A' && word.charAt(i) <= 'Z'){
                return false;
            }
        }
        return true;
    }
}

125. 验证回文串

题目要求

如果在将所有大写字符转换为小写字符、并移除所有非字母数字字符之后，短语正着读和反着读都一样。则可以认为该短语是一个 回文串 。

字母和数字都属于字母数字字符。

给你一个字符串 s，如果它是 回文串 ，返回 true ；否则，返回 false 。

示例 1：

输入: s = "A man, a plan, a canal: Panama"
输出：true
解释："amanaplanacanalpanama" 是回文串。

示例 2：

输入：s = "race a car"
输出：false
解释："raceacar" 不是回文串。

示例 3：

输入：s = " "
输出：true
解释：在移除非字母数字字符之后，s 是一个空字符串 "" 。
由于空字符串正着反着读都一样，所以是回文串。

提示：

  • 1 <= s.length <= 2 * 10^5
  • s 仅由可打印的 ASCII 字符组成

个人解答

  1. 先把字符串转为小写，然后从两头遍历字符串
  2. 如果前面遍历的当前字符不是数字或字母，就跳过，后面同理
  3. 如果两头遍历的当前字符不相同，就说明不是回文串，返回false
  4. 遍历完也没返回false说明是回文串，返回true
class Solution {
    public boolean isPalindrome(String s) {
        s = s.toLowerCase();
        int i = 0, j = s.length() - 1;
        while (i < j) {
            if(!(s.charAt(i) >= '0' && s.charAt(i) <= '9') && !(s.charAt(i) >= 'a' && s.charAt(i) <= 'z')){
                i++;
            }
            else if(!(s.charAt(j) >= '0' && s.charAt(j) <= '9') && !(s.charAt(j) >= 'a' && s.charAt(j) <= 'z')){
                j--;
            }
            else {
                if (s.charAt(i) != s.charAt(j)) return false;
                i++;
                j--;
            }
        }
        return true;
    }
}

14. 最长公共前缀

个人解答

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1：

输入：strs = ["flower","flow","flight"]
输出："fl"

示例 2：

输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。

提示：

  • 1 <= strs.length <= 200
  • 0 <= strs[i].length <= 200
  • strs[i] 仅由小写英文字母组成

个人解答

  1. 如果数组长度为0，说明没有字符串，最长公共前缀就是“”
  2. 然后把第一个字符串视为最长前缀，然后依次与后面的字符串比较
  3. 找到最长前缀和要比较的字符串的最长前缀，赋给最长前缀，最后返回即可
class Solution {
    public String longestCommonPrefix(String[] strs) {
        if(strs.length == 0){
            return "";
        }
        String maxPre = strs[0];
        for(int i=1;i<strs.length;i++){
            String s;
            int j=0;
            for (;j<Math.min(strs[i].length(),maxPre.length());j++){
                if(strs[i].charAt(j) != maxPre.charAt(j)){
                    break;
                }
            }
            if(j == 0){
                return "";
            }
            maxPre = strs[i].substring(0,j);
        }
        return maxPre;
    }
}

434. 字符串中的单词数

题目要求

统计字符串中的单词个数，这里的单词指的是连续的不是空格的字符。

请注意，你可以假定字符串里不包括任何不可打印的字符。

示例:

输入: "Hello, my name is John"
输出: 5
解释: 这里的单词是指连续的不是空格的字符，所以 "Hello," 算作 1 个单词。

个人解答

  1. 将字符串以空格分割成字符串数组
  2. 遍历分割的数组，如果当前字符串不是空串，就说明是一个单词，计数器加一
  3. 最后返回单词数量
class Solution {
    public int countSegments(String s) {
        String[] ss = s.split(" ");
        int count = 0;        
        for(int i=0;i<ss.length;i++){
            ss[i] = ss[i].replace(" ","");
            if(!ss[i].isEmpty()){
                count++;
            }
        }
        return count;
    }
}
